Using Git

Jun 26, 2022 · 54 words · 1 minute read

git 常用指令 🔗

    • objective 开启提示

  • 提交 git commit -m ""

  • 创建分支 git branch <分支名>

  • 切换分支 git checkout <分支名>

  • 合并分支

    1. git merge <分支名>

    2. git rebase <分支名>

    • 注意：先确定当前所处分支，再进行合并（用*标记）
    • 区别：merge 以当前分支为主，合并另一分支；rebase 以合并分支为主，将当前分支合并到合并分支中；

  • 指定提交记录(希哈值) git checkout <branch>

  • 相对引用

    1. git checkout HEAD^ j将HEAD向上移动一位
    2. git checkout HEAD~4 移动四位
    3. git branch -f main HEAD~3 将 main 分支强制指向 HEAD 的第 3 级父提交
